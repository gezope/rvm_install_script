Rails install on Ubuntu by gezope

Shortest way to install Ruby on Rails on Ubuntu

We found i was not enough easy to start riding on Rails on Ubuntu. Mostly not for beginners. Now it is easy! The shortest way to install Ruby on Rails 3 with RVM (Ruby Version Manager), Ruby1.9 is this install script. Later you can add more Ruby versions (including Rubinius or JRuby), other Rails or Gem versions or create new Gemsets for your projects. All later dependency issues solved. All packages included. Please let us know if any further improvements are necessary or if you've any problem with running this script.

Dependencies

Ubuntu Linux, installed Git, open a terminal - and patience:)

Install

- git clone git://github.com/davidpgero/rvm_install_script.git - cd rvm_install_script - chmod +x rvm_install_script.sh - ./rvm_install_script.sh
